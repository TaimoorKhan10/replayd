# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| 0.1.x (latest) | ✅ |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report them privately through one of these channels:

- **GitHub Security Advisories** — [Report a vulnerability](https://github.com/TaimoorKhan10/replayd/security/advisories/new) (preferred)
- **Email** — taimoorkhaniajaznabi@gmail.com

Include as much detail as you can: what the vulnerability is, how to reproduce it, and what impact it could have. We will respond within 72 hours and work with you to resolve it before any public disclosure.
